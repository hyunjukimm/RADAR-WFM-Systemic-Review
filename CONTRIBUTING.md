# Contributing

Thanks for contributing to the wearable foundation model survey.

## How To Contribute

- Add papers with enough metadata to support later comparison.
- Keep summaries concise and evidence-based.
- Prefer consistent naming for datasets, tasks, and model families.
- When possible, include links to official papers, code, and project pages.

## Pull Requests

Please:

- Describe what was added or changed
- Explain why it matters for the survey
- Keep formatting consistent with existing files
- Avoid duplicate entries

## Suggested Contribution Areas

- New paper entries
- Taxonomy updates
- Dataset and benchmark summaries
- Comparison tables
- Figure improvements

## Review Standard

Changes should improve clarity, accuracy, and organization of the survey.
